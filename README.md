# Coffee

Application for finding the nearest coffee shops.

## Technology stack
* UIKit
* VIPER
* SnapKit
* Yandex MapKit
* CoreLocation
* Alamofire

## Build locally
1. Clone the repository:
```
git clone https://github.com/oleegk/Coffee.git
```
2. Execute the following command in your project's directory to install dependencies:
```
pod install
```
